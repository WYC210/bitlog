🚀 快速开始

只需三步，即可集成到你的博客：


<link rel="stylesheet" href="/ui/widgets.css" />
<link rel="stylesheet" href="/ui/widgets-enhanced.css" />


<aside>
  <div class="widget">
    <div class="widget-header">
      <h3 class="widget-title">天气</h3>
    </div>
    <div class="widget-body">
      <!-- 组件内容 -->
    </div>
  </div>
</aside>


<script src="/ui/widgets.js"></script>

📦 文件结构

apps/web/public/
├── ui/
│   ├── base.css              # 基础样式
│   ├── widgets.css           # 组件样式
│   ├── widgets-enhanced.css  # 增强样式
│   └── widgets.js            # 组件逻辑
└── _templates/
    └── home.html             # 首页模板（已集成）

🔧 API 配置

在 widgets.js 中配置你的 API：

// 天气 API
async function fetchWeather() {
  const response = await fetch('YOUR_WEATHER_API_URL');
  return await response.json();
}

// GitHub API
async function fetchGitHubTrending() {
  const response = await fetch('https://api.github.com/...');
  return await response.json();
}

// 新闻 API
async function fetchNews() {
  const response = await fetch('YOUR_NEWS_API_URL');
  return await response.json();
}
