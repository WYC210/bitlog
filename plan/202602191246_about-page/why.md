# Why：新增「关于我」页面（/about）

## 背景

当前站点主导航为「首页/文章/项目/工具中心」，缺少一个对外展示个人信息与站点特色的聚合页。旧站 `newechoes_gitee` 已有 About 页的关键模块（技术栈 + 旅行足迹 3D 地球），可迁移复用思路与资源。

## 目标

- 新增公开页面 `/about`，并加入主导航：`首页 / 文章 / 项目 / 工具中心 / 关于我`
- 页面模块（从上到下建议）：
  1) **实时天气**：基于访问者 IP 自动定位（不弹授权），展示接口返回的所有字段
  2) **今日快讯**：每日新闻图片（图片）+ 程序员历史事件（JSON 列表）
  3) **我的技术栈**：后台可编辑
  4) **旅行足迹**：沿用 `newechoes_gitee` 的 3D 世界地图（Three.js + WASM + 地图数据），在浏览器侧运行
  5) **我的过往**：后台可编辑（时间线/经历）

## 非目标（本期不做）

- 不引入 React/Vite 打包体系到 Web 前台（选择方案 A：纯原生 JS 模块运行交互逻辑）
- 不将 3D 地球做“服务端渲染为可交互内容”（Worker 无 WebGL/DOM 环境，不可行）
- 不做复杂 CMS（本期仅在现有 settings 体系上扩展 about 配置）

## 成功标准（验收）

- 导航出现「关于我」，访问 `/about` 页面布局正常
- 实时天气无需授权自动加载，且把返回字段完整展示（含原始 JSON）
- 今日快讯：新闻图片可见；历史事件 events 列表可见
- 后台修改「技术栈/旅行足迹地点/我的过往」配置，刷新 `/about` 后立即生效
- 3D 地球可加载 maps + wasm 资源，交互（旋转/缩放/hover）工作正常
