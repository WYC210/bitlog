# Admin UI demos

Static HTML demos for comparing Admin sidebar + page layouts across 4 styles.

## Styles (multi-page)

- `docs/admin-ui-demos/classic/` (classic admin)
- `docs/admin-ui-demos/terminal/` (terminal / cyber)
- `docs/admin-ui-demos/glass/` (glassmorphism)
- `docs/admin-ui-demos/brutal/` (neo-brutalism)

Each folder contains:

- `posts.html` (posts list)
- `editor.html` (editor + preview)
- `settings.html` (settings with secondary nav)
- `account.html` (account / password / prefs)
- `login.html` (login)

Shared assets:

- `docs/admin-ui-demos/_shared/base.css`
- `docs/admin-ui-demos/_shared/app.js`

## Quick open (Windows)

```powershell
start docs\admin-ui-demos\classic\posts.html
start docs\admin-ui-demos\terminal\posts.html
start docs\admin-ui-demos\glass\posts.html
start docs\admin-ui-demos\brutal\posts.html
```
