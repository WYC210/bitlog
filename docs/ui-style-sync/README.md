# Bitlog UI 样式同步预览

这些 HTML 仅用于审阅设计方向，不会改动现有模板。

## 文件清单
- `web-home.html`：前台首页
- `web-home.dark.html`：前台首页（夜间）
- `web-articles.html`：前台文章列表
- `web-articles.dark.html`：前台文章列表（夜间）
- `web-post.html`：前台文章详情
- `web-post.dark.html`：前台文章详情（夜间）
- `admin-login.html`：后台登录
- `admin-login.dark.html`：后台登录（夜间）
- `admin-posts.html`：后台文章列表
- `admin-posts.dark.html`：后台文章列表（夜间）
- `admin-editor.html`：后台编辑器
- `admin-editor.dark.html`：后台编辑器（夜间）
- `admin-settings.html`：后台设置
- `admin-settings.dark.html`：后台设置（夜间）
- `admin-account.html`：后台账号
- `admin-account.dark.html`：后台账号（夜间）

## 预览方式
双击打开任意 HTML 文件，或在浏览器中拖拽打开。
如需主题切换，点击页面右上角“主题”按钮。
为核验背景与过渡效果，建议对照原博客截图关注：
- light/dark 的整体背景渐变层级是否连贯（无中间断层）
- 全局光晕边框与主题切换径向过渡是否存在
