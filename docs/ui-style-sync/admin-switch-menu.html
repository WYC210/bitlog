设置（总览，Demo）
设置页拆分：Tools / Projects / About / 站点配置（独立路由 + DOM 切换）。
为什么要拆分？
设置内容多、字段多，如果全塞在一个页面里，快捷键驱动会变得很痛苦。把每一块拆成独立路由， SWITCH 菜单就能像「主入口 → 子入口」一样工作：呼出、左右选、回车跳转，形成肌肉记忆。
推荐绑定（示例）：
Alt+S → 设置总览 Alt+T → Tools Alt+P → Projects Alt+A → About Alt+C → 站点配置
最近改动（示例）
模块 内容 时间
站点配置 更新 embed allowlist 今天
Projects 调整 maxItemsPerPlatform 昨天
