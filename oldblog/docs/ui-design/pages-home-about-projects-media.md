# 页面文档（首页 / 关于 / 项目 / 电影 / 书单 / 404）

## 首页（/）

- 居中单屏展示站点标题（大字号渐变文字）。
- 仅呈现品牌氛围，无其他模块内容。

## 关于（/about）

结构由两个主要区域构成：

1) 倒计时区块  
   - 标题：“距离退休还有”  
   - 卡片容器（圆角 + 阴影），内置倒计时组件  

2) 旅行足迹区块  
   - 标题：“我的旅行足迹”  
   - 卡片容器内嵌 3D 世界地图

## 项目（/projects）

- 标题区：显示平台名称与用户名。
- 内容区：瀑布流卡片展示项目列表。
- 卡片信息包含：头像、项目名、描述、语言、Star、Fork、更新时间。

## 电影（/movies）

- 页面标题：我看过的电影  
（中文文案，强调个人收藏）
- 列表以网格卡片展示，封面图为主视觉。

## 书单（/books）

- 页面标题：我读过的书  
（中文文案，强调个人收藏）
- 列表以网格卡片展示，封面图+书名+作者。

## 404 页面

- “404” 大号渐变字。
- 文案说明页面不存在。
- 提供返回首页按钮。
