# 动效与页面转场

## 主题切换动效

- 主题切换使用“扩散/收缩”过渡动画（点击点为中心）。
- 带波纹动画与图标切换；过渡时避免闪烁。
- 支持系统主题跟随与本地存储记忆。

## 页面转场（Swup）

- 页面切换使用淡入/淡出过渡。
- 文章页采用 `#article-content` 作为局部过渡容器，减少面包屑区域重绘。
- 带加载动画：圆环旋转，居中显示。
- 支持预加载与脚本重执行，确保切换后交互可用。

## 导航与菜单动效

- 桌面端导航高亮背景平滑过渡（位置/尺寸/透明度）。
- 移动端子菜单展开：高度、透明度与位移动画。
- 汉堡菜单三线变形为“X”。

## 文章页动效

- 阅读进度条随滚动填充。
- 返回顶部按钮随滚动显隐与平滑滚动。
- 目录（TOC）展开/收起与当前标题高亮。
