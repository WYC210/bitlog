# 页面文档（文章目录 / 文章详情 / 筛选页）

## 文章目录（/articles）

布局结构：

- 顶部面包屑条（可切换“筛选 / 网格”视图）
- 内容区域为网格卡片（目录/文章统一样式）

功能说明：

- **目录卡片**：展示子目录名称、子目录/文章数量。
- **文章卡片**：标题 + 摘要 + 日期 + “阅读全文”入口。
- **返回上级目录**：当路径非根目录时展示。
- **空目录提示**：目录无内容时提示。

## 文章详情（/articles/[...id]）

布局结构：

- 顶部进度条（随滚动更新）
- 面包屑区域（含返回列表按钮）
- 文章内容区（标题 / 日期 / 标签 / 正文）
- 相关文章推荐（同标签 / 同目录 / 最新）
- 右侧固定目录（TOC）
- 返回顶部按钮

功能说明：

- **过期提示**：文章超过指定天数显示提醒。
- **标签跳转**：点击标签进入筛选页。
- **目录 TOC**：根据标题生成分级目录，并高亮当前阅读位置。

## 文章筛选（/filtered）

布局结构：

- 顶部面包屑条（筛选与网格切换）
- 筛选控件（标签 / 时间范围 / 排序 / 每页数量）
- 结果区域（文章卡片网格）
- 分页控件

功能说明：

- 标签筛选与快速清除。
- 日期范围选择（支持快捷区间）。
- 结果统计与分页。
