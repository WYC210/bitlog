# 交互组件

## 搜索（Search）

布局特征：
- 输入框前置搜索图标，右侧包含清除与补全提示。
- 结果面板为下拉浮层，包含统计与列表。

交互特征：
- 支持搜索建议与自动补全。
- 支持键盘上下切换建议，Tab 键补全。
- 结果支持无限加载与“加载更多”。
- 点击结果后关闭面板并跳转。

## 面包屑（Breadcrumb）

布局特征：
- 左侧路径分段，右侧视图切换按钮。
- 移动端自动折叠中间路径，保留首尾。

交互特征：
- 视图切换：筛选 / 网格。
- 文章详情页提供“返回文章列表”按钮。

## 目录 TOC（文章详情）

布局特征：
- 右侧固定面板（大屏显示）。
- 支持多级标题折叠/展开。

交互特征：
- 当前阅读位置自动高亮。
- 点击条目平滑滚动。

## 阅读进度条

- 固定顶部细条，随滚动进度更新。

## 返回顶部

- 滚动超过阈值显示。
- 点击平滑回到顶部。
