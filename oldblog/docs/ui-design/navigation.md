# 导航体系

## 主导航结构

主导航采用分组结构，定义于 `NAV_STRUCTURE`：

- 首页（Home）
- 艺术（Art）
  - 观影（Movies）
  - 读书（Books）
- 文章（Articles）
  - 筛选（Filtered）
  - 网格（Articles）
- 其他（Other）
  - 关于（About）
  - 项目（Projects）

> 导航项按“主导航 + 子菜单”组织，子菜单为二级导航链接。

## 桌面端导航

- 固定顶部导航条，品牌区在左，搜索框居中偏右，导航项在右。
- 主导航高亮：使用双层高亮背景（primary/secondary）。
- 子菜单展开后，子项与主项同步高亮。

## 移动端导航

- 顶部保留搜索按钮与菜单按钮（汉堡样式）。
- 菜单展开：完整导航树展示，含分组展开/收起动画。
- 搜索与菜单互斥显示（打开一个会关闭另一个）。

## 搜索入口

- 桌面端：导航条内置搜索框（固定显示）。
- 移动端：点击搜索按钮展开搜索面板。

## 主题切换入口

- 桌面端：导航右侧独立按钮。
- 移动端：菜单底部“切换主题”入口。
